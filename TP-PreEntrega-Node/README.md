# 🛒 TP PreEntrega - Node.js

Proyecto backend desarrollado con Node.js que permite gestionar productos utilizando la FakeStore API.

---

## 🚀 Funcionalidades

- Obtener todos los productos
- Obtener producto por ID
- Crear producto
- Eliminar producto
- Actualizar producto
